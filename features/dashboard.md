# Dashboard

The Dashboard is your home screen in Sciorex. It gives you a quick overview of your workspace and fast access to recent items.

## Overview

When you open Sciorex, the Dashboard shows:

- **Welcome message** with your workspace name
- **Quick stats** about your tickets, flows, and agents
- **Recent items** for fast navigation
- **Quick actions** to create new items

## Stats Cards

The dashboard displays summary statistics:

| Card | What it shows |
|------|---------------|
| **Tickets** | Total count with breakdown by status |
| **In Progress** | Active work items |
| **Flows** | Number of configured flows |
| **Agents** | Number of configured agents |

Each card shows the change since your last visit (e.g., "+2" for new items).

## Recent Items

The recent items list shows your most recently accessed:

- **Tickets** with current status
- **Chat sessions** with last message preview
- **Flows** with last run status
- **Agents** with description

Click any item to navigate directly to it.

## Quick Actions

Start new work quickly:

| Action | What it does |
|--------|--------------|
| **New Ticket** | Opens ticket creation form |
| **New Chat** | Starts a new conversation |
| **New Flow** | Opens the flow editor |
| **New Agent** | Opens agent creation wizard |

## Navigation

From the Dashboard, you can:

- Click **View All** on any section to see the full list
- Click individual items to open their details
- Use the sidebar to navigate to other views
- Use keyboard shortcuts (Ctrl+1 through Ctrl+8) to switch views

## Keyboard Shortcut

Press `Ctrl+1` (or `Cmd+1` on Mac) to return to the Dashboard from anywhere.
